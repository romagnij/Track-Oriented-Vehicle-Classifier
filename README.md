This project includes the development of a machine-learning system designed to classify 
vehicles as track-oriented based on their technical specifications. Using the UCI Automobiles 
dataset, the project applied performance metrics and a custom scoring system to classify 
vehicles, followed by SMOTE oversampling to correct a significant class imbalance (154 
non-track-oriented vs. 39 track-oriented). After data cleaning, two predictive models were 
trained, with the Random Forest achieving strong performance metrics, including 0.85 
accuracy, 0.92 ROC-AUC, 0.83 precision, and 0.86 recall. 
The project offers a complete machine-learning pipeline, visualizations such as ROC curves, 
feature importance plots, and an interactive prediction tool capable of evaluating track-oriented 
vehicles based on user specifications. Overall, the project shows that engineered performance 
features combined with modern machine-learning techniques can effectively identify 
track-oriented vehicles and provide a foundation for future automotive analysis tools.
